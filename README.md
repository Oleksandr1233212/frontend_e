# **Frontend LMS System**

Система керування навчанням (**Learning Management System**), створена на базі **Vue 3**. Цей проект включає панель адміністратора, особисті кабінети викладачів і студентів, а також публічні сторінки для відображення курсів, відгуків і контактної інформації.

---

## **Вимоги**

- **Git** – для роботи з репозиторієм.  
- **Node.js** – версія 16.x або вище.  
- **npm** – для управління залежностями.  
- **Vite** – для локального серверу розробки.

---

## **Встановлення**

1. **Клонування репозиторію**  
   Виконайте команду для копіювання проекту:

   ```bash
   git clone https://github.com/Oleksandr1233212/frontend_e.git
   cd frontend_e


2. Встановлення залежностей
    Встановіть усі необхідні залежності:

npm install

3. Запуск проекту в режимі розробки
    npm run dev

4. Перегляд у браузері
    http://localhost:5173

Структура проету :
src/

│   ├── common/           # Загальні компоненти (Header, Footer тощо)
│   │   ├── Header.vue    # Заголовок додатку
│   │   ├── Footer.vue    # Футер додатку
│   │   ├── Sidebar.vue   # Бокова панель
│   ├── ui/               # Компоненти інтерфейсу
│   │   ├── Button.vue    # Загальна кнопка
│   │   ├── Modal.vue     # Модальне вікно
│   ├── forms/            # Компоненти для форм
│   │   ├── LoginForm.vue # Форма входу
│   │   ├── RegisterForm.vue # Форма реєстрації
│   ├── admin/            # Компоненти для адміністратора
│   │   ├── AdminCard.vue # Карточка адміністратора
│   │   ├── AdminTable.vue# Таблиця адміністратора
│   ├── teacher/          # Компоненти для викладача
│   │   ├── TeacherCard.vue # Карточка викладача
│   │   ├── CourseTable.vue # Таблиця курсів
│   ├── student/          # Компоненти для студента
│       ├── AssignmentCard.vue # Карточка завдань
│       ├── ProgressChart.vue  # Графік прогресу

│   ├── useStorage.js     # Хук для роботи зі сховищем
├── layouts/              # Глобальні макети сторінок
│   ├── AppAdmin.vue      # Макет для адміністратора
│   ├── AppTeacher.vue    # Макет для викладача

│   ├── AppUnauth.vue     # Макет для неавторизованих користувачів
├── pages/                # Сторінки додатку
│   ├── auth/             # Сторінки аутентифікації
│   │   ├── Login.vue     # Сторінка входу
│   │   ├── Register.vue  # Сторінка реєстрації
│   ├── admin/            # Сторінки для адміністратора
│   │   ├── Dashboard.vue # Головна сторінка адміністратора
│   │   ├── ManageUsers.vue # Керування користувачами
│   │   ├── ManageCourses.vue # Керування курсами
│   ├── teacher/          # Сторінки для викладача
│   │   ├── Dashboard.vue # Головна сторінка викладача

│   │   ├── StudentProgress.vue # Прогрес студентів

│       ├── Assignments.vue # Завдання

├── stores/               # Pinia стори для глобального стану

│   ├── userStore.js      # Pinia стор для профілю користувача
├── App.vue               # Головний компонент
├── main.js               # Точка входу в додаток
├── .env                  # Змінні середовища (API URL тощо)
├── vite.config.js        # Конфігурація Vite
└── package.json          # Інформація про залежності
